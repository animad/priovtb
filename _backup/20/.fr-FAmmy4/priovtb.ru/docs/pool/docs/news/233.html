Прио-Внешторгбанк занял 76 место в рейтинге РБК по объему выданных потребительских кредитов в I полугодии 2010 года. За это время было выдано беззалоговых кредитов на сумму 54,16 млн. рублей. Прирост по сравнению с показателями первого полугодия 2009 года составил 269,68%. Программа потребительского кредитования разработана для физических лиц в рамках зарплатных проектов по пластиковым картам банка. Информацию по всем предоставляемым кредитам вы можете найти на нашем сайте или получить по телефону 44-54-25.
