Новые Бланки Паспортов сделки, Справки о валютной операции, Заявлений, Справки о подтверждающих документах размещены на сайте Банка в разделе «Услуги для Вашего бизнеса» -> «Международные расчеты и валютных контроль» (http://priovtb.com/?dr=megdu.php). Для получения бланков документов по системе «Клиент-Банк» просьба обратиться к сотруднику банка, осуществляющему валютный контроль. Для передачи в банк документов валютного контроля по системе «Клиент-Банк» требуется обновить АРМ «Клиент-Банк» по валютным счетам. Дистрибутив новой версии АРМ «Клиент-Банк» доступен на сайте банка в разделе «Услуги для Вашего бизнеса» -> «Программное обеспечение по системе «Клиент-Банк»» (http://priovtb.com/?dr=kb_prog.php). Обращаем также Ваше внимание, что в новой версии системы «Клиент-Банк» изменена форма документа «Валютное платежное поручение». Рекомендации по заполнению размещены на сайте банка в разделе «Программное обеспечение по системе «Клиент-Банк»» (http://priovtb.com/?dr=kb_prog.php). За более подробной информацией просьба обращаться в Службу технической поддержки по телефону: (4912) 50-44-50. При наличие технических проблем с обновлением АРМ «Клиент-Банк» до новой версии Банк в период с 1 — 5 октября 2012 года предоставляет Вам возможность передавать документы валютного контроля следующим образом: 1.Вы заполняете бланк документа, размещенный на сайте банка или высланный сотрудником банка. Сохраняете документ, указав в имени файла тип документа (паспорт сделки по контракту, паспорт сделки по кредитному договору, справка о валютных операциях, справка о подтверждающих документах, заявление). 2.В АРМ «Клиент-Банк» по валютным счетам входите в пункт меню «Валютные документы» -> «Валютные документы в банк» и выбираете кнопку «Загрузить документ из файла». 3.После этого выполняете отправку документа в банк в обычном порядке.
